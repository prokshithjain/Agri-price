A Machine Learning Model to Predict Student Academics Course Interest

Education system plays a vital role in any ones carrier. Selecting the suitable course for the final year degree students is a challenging task in the current education sector. In some educational institutes carrier counseling is done to know the students area of interest in higher course selection. Career counseling is more about knowing and understanding about students and their capabilities and abilities. In the final year degree, students think about selecting the suitable course (higher studies) based on their interest. It is this time every student gets a lot of guidance from various circles (parents, teachers, other educational specialists, etc.). Course selection of the students depends on so many education factors like semester results, gender factor, age factor, attendance factor etc. In this proposed system we develop automation for Course recommendation for the final year students. Course recommendation is done using Machine Learning algorithms. Efficient ML algorithm used to process the educational data and Course recommendation is done. Proposed system is a browser based application meant for a college developed using Microsoft technologies such as Visual Studio Code and PYTHON
