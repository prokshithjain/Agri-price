# Agriculture Commodities Price Prediction

## Project Overview

This project predicts the prices of agricultural commodities such as pulses and vegetables using Machine Learning algorithms like Random Forest and KNN.

## Features

- Pulse price prediction
- Vegetable price prediction
- Random Forest model
- KNN model
- Web-based user interface using Flask
- Dataset visualization and comparison

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap
- Machine Learning
- Random Forest
- KNN
- MySQL

## Installation

1. Clone the repository:
   git clone https://github.com/prokshithjain/Agri-price.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py
